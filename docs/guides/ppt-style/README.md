# PPT Style Guidelines Archive

This index collects the presentation-generation style references used by FacetDeck.

## Documents

- Animation patterns: `guidelines/ppt-style/animation-patterns.md`
- HTML presentation template: `guidelines/ppt-style/html-template.md`
- Style presets catalog: `guidelines/ppt-style/STYLE_PRESETS.md`
- Base viewport CSS: `guidelines/ppt-style/viewport-base.css`

## Recommended Reading Order

1. `html-template.md`
2. `animation-patterns.md`
3. `STYLE_PRESETS.md`
4. `viewport-base.css`

## Usage

- Use these docs as generation constraints and quality guardrails.
- Keep "core text remains visible" as a non-negotiable rule for animation design.
- Prefer abstract-shape design language over generic illustration-heavy outputs.

## Maintenance Notes

- When guideline files are updated in `guidelines/ppt-style/`, update this index if links or structure change.
